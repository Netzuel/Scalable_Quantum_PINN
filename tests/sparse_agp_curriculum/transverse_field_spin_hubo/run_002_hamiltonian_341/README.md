# run_002 / hamiltonian_341

This study uses a tracked snapshot of
`bitstrings_operator/datasets/HAMILTONIANS_SPIN/run_002/hamiltonian_341`.
It has 24 qubits and 216 terms: 24 one-body, 72 two-body, and 120 three-body
spin interactions.

The source minimum came from simulated annealing. The benchmark independently
evaluates all `2**24` computational-basis assignments and confirms a unique
exact ground state. Source files, conversion metadata, and SHA-256 provenance
are stored under `q24/sweep_test/problem_data/`.
